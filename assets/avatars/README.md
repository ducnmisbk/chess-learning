# User Avatars

Place user profile avatar options here.

Required: 8-12 animal avatars in SVG format
Suggested animals:
- bear, cat, dog, owl, fox, panda
- rabbit, lion, penguin, koala, elephant, giraffe

Format: avatar-{number}-{animal}.svg
Example: avatar-01-bear.svg

See ASSETS_ORGANIZATION_GUIDE.md for design guidelines.
